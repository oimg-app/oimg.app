## Component-Level Interaction Contracts

### CENTER-01: CenterPane shell

| Requirement | Spec |
|-------------|------|
| Takes full available height | `h-full` on root div; parent AppShell provides flex context |
| 3 vertical sections | `flex flex-col` with `shrink-0` header + `flex-1 min-h-0` stage + `shrink-0` delta |
| No scroll on root | `overflow-hidden` — only image-frame clips internally |

### CENTER-02: Center header

| Interaction | Behavior |
|-------------|----------|
| File selected | All breadcrumb tags re-render from `$selectedFile` |
| `file.q === null` | q-tag is omitted entirely (conditional render) |
| Zoom popover trigger | Ghost button, height 24px, `Eye` icon + current zoom value + `CaretDown` |
| Zoom option click | Calls `setZoom(value)` where value is numeric (25, 50, 100, 200) or string `'fit'`; closes popover |
| Zoom active indicator | Selected item shows checkmark (shadcn Popover item with check pattern matching Phase 4 `.pi.check.on`) |
| No file selected | Renders empty breadcrumb: just `Queue` span; "Optimized" pill hidden |

### CENTER-03: Compare stage

| Interaction | Behavior |
|-------------|----------|
| Initial split | Reads `uiAtom.split` (default 50) |
| Drag split handle | `onMouseDown` on `.split-handle`; `mousemove` on window calculates new pct; calls `setSplit(pct)` clamped `[2, 98]`; `mouseup` removes listeners |
| CSS --split applied | Inline style `{ '--split': `${split}%` }` on `.image-frame` div |
| Image layers | Placeholder colored divs (no real image in Phase 5); `layer-orig` = `bg-[var(--color-bg-2)]`; `layer-opt` = `bg-[var(--color-bg-3)]` |
| No file selected | Stage renders with default split=50 and empty labels |

### CENTER-04: Delta strip

| Requirement | Spec |
|-------------|------|
| Data sources | `$selectedFile` for file metrics; `settingsAtom` for codec/q/method |
| Stub metrics | SSIM, Butteraugli, Decode are hardcoded stub values — no computation in Phase 5 |
| No file selected | Cards render with `—` placeholder values; `.v` text remains `fg-3` |
| Savings card accent | `text-[var(--color-accent)] font-semibold` on `.v` span only |

### Zoom popover options

| Display | Value written to `uiAtom.zoom` |
|---------|-------------------------------|
| 25% | 25 |
| 50% | 50 |
| 100% | 100 |
| 200% | 200 |
| Fit | `'fit'` |

Default active: whichever matches `uiAtom.zoom` (initial default: 100).

## Critical Implementation Notes

1. **CSS custom property for split:** Set `style={{ '--split': `${split}%` } as React.CSSProperties}` directly on the `.image-frame` div. TypeScript requires the cast.

2. **`$selectedFile` null guard:** All three sub-components (`CenterHeader`, `CompareStage`, `DeltaStrip`) must handle `selectedFile === null`. Use `selectedFile ?? STUB_FILES[0]` pattern from Phase 2, or render empty/placeholder state.

3. **`settingsAtom` access in DeltaStrip:** Import `useStore` + `settingsAtom` from `@/stores/settings`. Read `codec`, `q`, `method` for the "Optimized" card subtitle. Never `useState`.

4. **Split drag — pointer capture:** Use `window.addEventListener('mousemove')` + `window.addEventListener('mouseup')` pattern (matches `app.jsx` reference exactly). Do not use `onPointerCapture` — the reference uses mouse events only.

5. **Phase attribution header:** All new files require `// Phase 05 — {REQUIREMENT_ID}: {description}` comment at top.

6. **`fmtBytes` / `fmtPct` import:** Always from `@/lib/format`, never inline or from stub-data.

7. **CompareStage layer colors (MVP placeholder):** `layer-orig` = `bg-[var(--color-bg-2)]`, `layer-opt` = `bg-[var(--color-bg-3)]`. Real image rendering is v2 scope.

8. **STORE-08 compliance:** Zero `useState` for data. Only `useStore(atom)` for data reads. Split drag is the only exception — use `useRef` for the drag state (not `useState`) to avoid re-renders during drag.

```typescript
import { enqueueRasterPreview } from '@/hooks/useBatchOrchestrate'
import { useRef, useCallback } from 'react'

// Inside InspectorPane component:
const debounceTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null)
const debouncedPreview = useCallback((fileId: string) => {
  if (debounceTimerRef.current) clearTimeout(debounceTimerRef.current)
  debounceTimerRef.current = setTimeout(() => {
    enqueueRasterPreview(fileId)
  }, 200)
}, [])
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| UI slider value → setPerFileCodec → adapter encode | User-controlled quality number (0–100 range) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-5-04-01 | Tampering | quality slider programmatically set outside 0–100 range | mitigate | Radix Slider min/max props enforce the range at UI level; adapter config builders clamp with Math.max/Math.min before passing to jSquash encode() |
| T-5-04-02 | Spoofing | FileEntry.format is an unknown value (not in 'svg'|'png'|'jpeg'|'webp'|'avif') | mitigate | CodecTabContent returns null for unknown format (fallthrough case) — no panel renders, no error thrown |
| T-5-04-03 | Information Disclosure | InspectorPane re-renders with stale file after setSelected | accept | React 19 renders synchronously after store mutation; stale reads are transient (< 1 frame); no security impact on client-side tool |
| T-5-04-04 | XSS | file.name displayed in CodecTabContent format indicator | mitigate | format pill shows `format` string (controlled union type), not file.name. File.name is NOT rendered in codec panels — only in FilesPane file row where existing escaping applies |
| T-5-04-05 | Denial of Service | Rapid slider drag fires enqueueRasterPreview on every tick | mitigate | 200ms debounce in debouncedPreview + enqueueRasterPreview calls cancelByPrefix('preview-') before enqueuing — same pattern as Phase 3 SVG live re-optimize |

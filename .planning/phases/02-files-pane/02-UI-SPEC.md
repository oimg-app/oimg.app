## Component Design Contract

### 1. FilesPane Header (FILES-01)

- Height: 32px (`h-8`)
- Background: `bg-[var(--bg-1)]`
- Border bottom: `border-b border-[var(--line)]`
- Padding: `px-3` (12px)
- Label: "QUEUE · N FILES" — `font-mono text-[10.5px] font-semibold tracking-[0.08em] uppercase text-[var(--fg-2)]`
- N = `$filteredFiles.length` from nanostores computed atom
- Icon buttons (sort + add): `w-[22px] h-[22px]` — `text-[var(--fg-2)] hover:bg-[var(--bg-3)] hover:text-[var(--fg-0)]` — `rounded`
- Sort icon: `Funnel` from @phosphor-icons/react (size 13)
- Add icon: `Plus` from @phosphor-icons/react (size 13)
- Sort popover trigger: shadcn `Popover`. Options: "Queue order / File size / Savings % / Name / Format". Each calls `setSortBy(s)`. Active option uses accent text.

### 2. Dropzone (FILES-02)

- Outer margin: 12px all sides (`m-3`)
- Padding: 14px (`p-[14px]`)
- Border: `border border-dashed border-[var(--line-strong)] rounded-[6px]`
- Text align: center
- Background: transparent (default); drag-over: `bg-[var(--accent-dim)] border-[var(--accent)] text-[var(--fg-0)]`
- Primary label: "Drop images to optimize" — `text-[13px] font-semibold text-[var(--fg-0)] block mb-[3px]`
- Secondary label: "or click to browse · max 200 files" — `text-[12px] text-[var(--fg-2)]`
- Format pills row: "SVG · PNG · JPEG · WEBP · AVIF · JXL" — `font-mono text-[10px] font-medium tracking-[0.05em] text-[var(--fg-3)] mt-[6px]`
- Always visible above file list (not a conditional state)

### 3. FileRow (FILES-03)

Grid: `grid-cols-[28px_1fr_auto] gap-[10px] items-center px-3 py-[7px] border-b border-[var(--line)] relative cursor-default`

- Default: `hover:bg-[var(--bg-1)]`
- Selected: `bg-[var(--accent-dim)]` + left border stripe: `before:absolute before:left-0 before:top-0 before:bottom-0 before:w-[2px] before:bg-[var(--accent)]`
- Click: calls `selectFile(id)`
- Selected state: read from `filesAtom.selectedId`

#### 3a. Format Badge (thumb)

Size: `w-7 h-7` (28×28px). Border: `border border-[var(--line)] rounded-[3px]`. Center content via `grid place-items-center`. Text: `font-mono text-[8.5px] font-semibold`.

| Format | Tailwind background | Text color |
|--------|--------------------|-----------:|
| svg | `bg-[repeating-linear-gradient(45deg,var(--bg-2)_0_4px,var(--bg-3)_4px_5px)]` | `text-[var(--accent)]` |
| png | `bg-[linear-gradient(135deg,oklch(0.55_0.12_250)_0%,oklch(0.45_0.10_280)_100%)]` | `text-[oklch(0.95_0_0)]` |
| jpg | `bg-[linear-gradient(135deg,oklch(0.65_0.13_60)_0%,oklch(0.55_0.15_30)_100%)]` | `text-[oklch(0.95_0_0)]` |
| webp | `bg-[linear-gradient(135deg,oklch(0.60_0.14_195)_0%,oklch(0.50_0.12_220)_100%)]` | `text-[oklch(0.95_0_0)]` |
| avif | `bg-[linear-gradient(135deg,oklch(0.60_0.14_320)_0%,oklch(0.45_0.12_290)_100%)]` | `text-[oklch(0.95_0_0)]` |

Badge text: uppercase abbreviation (`file.type.toUpperCase()` — 3-4 chars max).

#### 3b. File Meta

- File name: `text-[12px] font-medium text-[var(--fg-0)] truncate` (overflow ellipsis, no wrap)
- Stat row: `font-mono text-[11px] text-[var(--fg-2)] flex gap-[6px] items-center mt-[2px]`
  - Orig size: `fmtBytes(file.orig)`
  - Arrow: `→` — `text-[var(--fg-3)]`
  - Opt size: `fmtBytes(file.opt)`
  - Savings badge: `font-semibold` — accent when savings ≥ 30%: `text-[var(--accent)]`; warn when savings < 30%: `text-[var(--warn)]`; value from `fmtPct(file.orig, file.opt)`

#### 3c. Row Right Column

Flex column, items centered:
- Status dot: `w-2 h-2 rounded-full flex-shrink-0`
  - queued: `bg-[var(--fg-3)]`
  - processing: `bg-[var(--info)] animate-pulse` (pulse keyframe: opacity 0.4→1→0.4, 1.4s ease-in-out infinite)
  - done: `bg-[var(--accent)]`
  - error: `bg-[var(--err)]`
- Context button (ctxbtn): `w-[22px] h-[22px] grid place-items-center rounded text-[var(--fg-2)] hover:bg-[var(--bg-3)] hover:text-[var(--fg-0)]`
  - Icon: `DotsThreeVertical` size 12
  - On click: dispatch programmatic `contextmenu` event on `rowRef`

#### 3d. Progress bar (when `file.status === 'processing'` and `file.prog` defined)

Full-width, below the stat row. Height: `h-[2px]`. Background track: `bg-[var(--bg-3)]`. Fill: `bg-[var(--info)]`. Width: `${file.prog}%`. Transition: `transition-[width] duration-150`.

Trigger: `ContextMenu` wrapping the `FileRow` div (via `ContextMenuTrigger asChild` — attach ref to trigger, not inner div).

Menu items in order:

| Copy data URI | `Copy` (13px) | `pushToast` stub | default |

| Reveal in compare | `Eye` (13px) | `pushToast` stub | default |

### 5. Totals Bar (FILES-05)

Position: fixed to bottom of pane, above scrollable area. `border-t border-[var(--line)] bg-[var(--bg-1)] px-3 py-[10px] grid grid-cols-2 gap-x-[14px] gap-y-1`

4 stat cells (2×2 grid):

| Cell | Label | Value source | Value color |
|------|-------|-------------|-------------|
| Total before | "BEFORE" | `fmtBytes($totals.orig)` | `text-[var(--fg-0)]` |

| Compression | "SAVED %" | `$totals.pct.toFixed(1) + '%'` | `text-[var(--accent)]` |

Label: `font-mono text-[10px] tracking-[0.06em] uppercase text-[var(--fg-2)]`
Value: `font-mono text-[13px] font-semibold tabular-nums`

## Accessibility Contract

- All icon-only buttons (`iconbtn`, `ctxbtn`) must have `aria-label`. Values: sort button = `"Sort files"`, add button = `"Add files"`, context button = `"File options"`.
- `ContextMenu` from Radix handles focus trap, keyboard nav (↑↓ keys, Enter to select, Escape to close), and `aria-haspopup` automatically.
- Status dot: add `aria-label` with status value (e.g., `aria-label="Status: done"`). Not focusable (`aria-hidden` if status is also implied by styling only — defer to WCAG audit in Phase 7).
- File row: `role="row"` or native `<li>` in `<ul>` (preferred). If using `<ul>/<li>`, the list needs `aria-label="File queue"`.
- Selected row: `aria-selected="true"` on the row element.
- Format badge: `aria-hidden="true"` — decorative, file type is also in the filename.
- Color contrast: all text on surfaces meets WCAG AA (4.5:1 for normal text, 3:1 for large). Verified by design — oklch palette was designed for contrast.

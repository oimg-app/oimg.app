## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-03 | Tampering | useOptimize codec mapping | mitigate | File `type` is normalized through an explicit lowercase→enum map; unknown types map to a value the worker re-validates against its 5-codec enum (defense in depth with Plan 02 T-08-03) — no dynamic property indexing |
| T-08-05 | Denial of Service | runtimeAtom stale `running` after batch | mitigate | `setJobCounts` derives `running = runningJobs>0||queuedJobs>0`; pool calls it on every drain/release so `running` returns to false exactly when the queue empties (RESEARCH.md Pitfall 3) |
| T-08-06 | Denial of Service | one stubbed-codec rejection aborting the batch | accept | `Promise.allSettled` isolates per-job failures; a NotImplemented throw for WebP/JPEG/AVIF/SVG in Phase 8 rejects only that job, pool still drains and counts reset |
| T-08-SC | Tampering | npm/pip/cargo installs | mitigate | No new packages installed (nanostores/@nanostores/react pre-existing; RESEARCH.md Package Legitimacy Audit) — no install step, no slopcheck gate |

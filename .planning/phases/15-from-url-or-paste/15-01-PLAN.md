<objective>
Create the `pickFromUrl(url)` dispatcher that backs ING-01's URL-string ingest path. The function fetches an arbitrary `http(s)` URL, validates content-type and size, derives a safe filename, and returns a `File` ready for `useIngest.ingest()`. Returns `null` on every failure and emits its own user-facing toast so callers never need a try/catch.

Purpose: SC-2 (URL ingest with CORS-honest failure messaging) and T-15-04 mitigation (sanitizeBaseName on the URL-derived filename).
Output: `src/lib/url-ingest.ts` + Node unit test + Playwright e2e.
</objective>

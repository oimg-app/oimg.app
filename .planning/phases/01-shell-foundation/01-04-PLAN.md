## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Theme toggle → localStorage | TitleBar toggle calls useTheme().toggle(); writes only 'dark' or 'light' to localStorage['oimg-theme'] |
| DOM attributes → ARIA tree | role/aria-label attributes expose structure to screen readers; must be semantically correct |
| Playwright → localhost:5173 | Test runner connects to local dev server only; no external network; isolated test environment |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | useTheme / localStorage | accept | localStorage value is constrained to ThemeMode ('dark'/'light'); invalid values fall back to 'dark'. No external data accepted. |
| T-04-02 | Information Disclosure | ARIA attributes | accept | ARIA roles and labels describe UI structure (not user data); no sensitive information in landmark labels |
| T-04-03 | Elevation of Privilege | placeholder panel components | accept | Placeholder divs have no interactive elements; no XSS surface in Phase 1 (no user content rendered) |
</threat_model>

|----------|-------------|
| `runtimeAtom.versions` + `runtimeAtom.caps` → JSX text children | Default React escaping; types enforce string/boolean/number |
| `JSON.stringify({versions, caps}, null, 2)` → clipboard write | Pretty-printed JSON; chokepoint handles toast surface |

| clipboard contents → user's external app (paste target) | Out of zero-server boundary; the user controls where they paste |

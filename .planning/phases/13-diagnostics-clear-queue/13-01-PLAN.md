| Boundary | Description |
|----------|-------------|
| `node_modules/<pkg>/package.json` → `vite.config.ts` `readVer()` | Build-time read of trusted dev-dep metadata; pinned + lockfile-controlled |
| `vite.config.ts` `define` → bundle JS literal | Vite substitutes the JSON.stringify'd value verbatim into emitted JS |
| `src/lib/versions.ts` → React JSX text | Type system enforces `string`; React's default text escaping handles malicious chars in package versions |

NO `process.env.*` literals are passed to `define`. Task 1 acceptance asserts `grep -c "process.env" vite.config.ts` does not regress. |
| T-13-04 | XSS via injected version string | `src/lib/versions.ts` → React JSX | mitigate | Type system constrains `BUILD_VERSIONS.svgo` / `jsquash.*` to `string`. Plan 04 (StatusBar) + Plan 07 (Diagnostics tab) render via React text children — default escaping. No `dangerouslySetInnerHTML` path through `versions`. JSON.stringify in vite.config.ts plus React's escaping are defense-in-depth. |

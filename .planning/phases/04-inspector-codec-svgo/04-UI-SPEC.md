## Component-Level Interaction Contracts

### INSP-01: Tab bar + auto-switch

| Interaction | Behavior |
|-------------|----------|
| User clicks tab | `setTab(t)` called; controlled `Tabs value={tab}` updates |
| SVG file selected | `useEffect` fires → `setTab('svgo')` |
| Non-SVG file selected while tab='svgo' | `useEffect` fires → `setTab('codec')` |
| tab='output' or 'report' | Renders stub placeholder; no crash |
| Dependency array | `[selectedFile?.id, selectedFile?.type]` — NOT `tab` (infinite loop prevention) |

### INSP-02: Output format section

| Element | Spec |
|---------|------|
| Codec buttons | `SegControl` spanning full width; options from `CODECS` constant; active = `settingsAtom.codec` |
| Lossless toggle | `Switch` + label "Lossless"; hidden when `codec === 'SVG'` |
| Section title | "Output format" |

### INSP-03: Parameters section

| Element | Spec |
|---------|------|
| Section visibility | Hidden when `codec === 'SVG'` |
| Section title | `{codec} parameters` |
| Section badge | `{ text: CODEC_ENGINE[codec] }` — no `acc` flag (default bg-2 chip) |
| Quality slider | Label "Quality", `min=0 max=100 step=1`, `value={[settings.q]}`, readout shows numeric |
| Effort slider | Label "Effort", `min=0 max=6 step=1`, `value={[settings.method]}`, readout shows numeric |
| Palette seg | Row "Palette" + `SegControl options={['off','auto','PNG-8']}` — shown ONLY when `codec === 'PNG'` |
| Subsample seg | Row "Subsample" + `SegControl options={['4:2:0','4:4:4']}` — shown ONLY when `codec === 'AVIF'` |

Codec engine badge map (hardcoded):
```
AVIF → 'libavif'
WebP → 'libwebp'
JPEG → 'mozjpeg'
PNG  → 'oxipng'
SVG  → 'svgo'   (section hidden anyway)
```

## Critical Implementation Notes

1. **Slider value type:** `<Slider value={[settings.q]} onValueChange={([v]) => setQuality(v)} />` — always array, never scalar.

2. **Circular ESM guard:** `src/stores/ui.ts` MUST NOT import from `src/stores/settings.ts`. Tab auto-switch logic lives in `InspectorPane.tsx` as a `useEffect`.

3. **useEffect dep array:** `[selectedFile?.id, selectedFile?.type]` only — do NOT include `tab` (causes infinite loop).

4. **Constants source:** Components import `CODECS`, `RESIZE_ALGS`, `FIT_MODES` from `@/stores/settings` (re-exported). Never import from `@/lib/stub-data` in components (STORE-08).

5. **STORE-08:** Zero `useState` for data. Only `useStore(atom)` for data reads. Ephemeral hover/focus via Tailwind state variants only.

6. **Phase attribution header:** All new files require `// Phase 04 — {REQUIREMENT_ID}: {description}` comment at top.

7. **OutputPanel / ReportPanel stubs:** Must exist as non-null TabsContent. Render a `<p className="p-3 text-xs text-[var(--color-fg-3)]">` placeholder.

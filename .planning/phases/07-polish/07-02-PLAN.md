  <name>Task 1: Fix theme FOUC and wire data-theme attribute</name>
  <files>index.html, src/components/shell/AppShell/AppShell.tsx</files>
  <action>
    Edit index.html — add an inline script in &lt;head&gt;, BEFORE the title tag and any CSS link. The script must be synchronous (no type="module"), so it runs before first paint:

    ```
    (function() {
      try {
        var t = localStorage.getItem('oimg-theme') || 'dark';
        document.documentElement.classList.toggle('dark', t === 'dark');
        document.documentElement.setAttribute('data-theme', t);
      } catch(e) {
        document.documentElement.classList.add('dark');
        document.documentElement.setAttribute('data-theme', 'dark');
      }
    })();
    ```

    Wrap in a &lt;script&gt; tag (no type attribute — must be classic script, not module).

    The existing class="dark" on &lt;html&gt; is a static fallback for no-JS. Keep it. The inline script will override it correctly when JS runs.

    The rule:
    ```css
    /* Global focus ring — WCAG AA — Phase 07-polish / Plan 02 */
    :focus-visible {
      outline: 2px solid var(--color-accent);
    ```
